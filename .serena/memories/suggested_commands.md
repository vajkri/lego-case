# Suggested Commands

## Development
- `npm run dev` ‚Äî Start Next.js dev server (hot reload)
- `npm run build` ‚Äî Production build (static export to `out/`)
- `npm start` ‚Äî Serve the static build (`serve out`)

## Quality
- `npm run lint` ‚Äî Run ESLint (via `next lint`)
- `npm test` ‚Äî Run Vitest test suite (`vitest run`)

## Git (macOS/Darwin)
- `git status` ‚Äî Check working tree status
- `git log --oneline -10` ‚Äî Recent commit history
- `git diff` ‚Äî Unstaged changes
- `git diff --cached` ‚Äî Staged changes

## System (Darwin)
- `ls`, `cat`, `find`, `grep` ‚Äî Standard Unix utilities (macOS variants)
- `open .` ‚Äî Open current directory in Finder
- `pbcopy` / `pbpaste` ‚Äî Clipboard access
