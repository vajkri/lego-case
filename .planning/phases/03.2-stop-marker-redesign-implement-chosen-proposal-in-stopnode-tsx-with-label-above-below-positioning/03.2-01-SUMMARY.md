## Decisions Made

- `labelPosition` is required (not optional) on the `Stop` type — TypeScript enforces that every stop has an explicit positioning decision, preventing accidental omissions in future stops
- The 5-state test spec uses `'default'` as the state name for unvisited stops, which differs from the current implementation's `'unvisited'`. The test is intentionally RED as a forcing function for Plan 02 to rename the state correctly.
