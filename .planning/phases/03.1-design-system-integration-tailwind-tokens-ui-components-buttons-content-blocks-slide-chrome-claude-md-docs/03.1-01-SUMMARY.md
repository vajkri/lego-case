## Decisions Made
- rgba() values in :root not @theme — Tailwind v4 @theme silently drops or mishandles rgba color values; tints and depth shadows must live in :root and be referenced via CSS custom property references in style props
- Button exports `className` prop for composition — downstream components (SlideNavArrows, PresentationFooter) can compose classes onto Button
- border-b-4 with active:border-b-2 + active:translate-y-0.5 for the brick press effect — pure CSS `active:` pseudo-class, no JS event handlers needed
