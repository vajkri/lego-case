## Decisions Made

- SlideFrame is NOT `'use client'` — layout-only, no hooks. SlideOverlay provides the client boundary.
- Nav arrows are inlined in SlideOverlay as ReactNode props to SlideFrame's `leftArrow`/`rightArrow` slots, not via SlideNavArrows.tsx import. SlideNavArrows.tsx was refactored to use Button and kept for potential reuse.
- SlideContent theme fix scoped to color classes only (`text-white` -> `text-lego-dark`, `bg-white/10` -> `bg-lego-grey-wash`) — Phase 4 owns structural redesign.
- Close button `id="slide-close-btn"` is an immutable constraint documented in SlideFrame source comments.
