Create `src/components/features/map/RoadPath.tsx`.

This file must be created FIRST (before MapSvg) because MapSvg imports from it.

**ROAD_PATH_D:** Design a winding SVG path `d` string that:
- Uses a coordinate space of `0 0 1400 800` (matching the MapSvg viewBox below)
- Starts at approximately `(80, 560)` — lower-left rural area
- Ends at approximately `(1320, 200)` — upper-right urban/cityscape area
- Has 3-4 significant curves (not straight, not just one curve — visually "pretty" per CONTEXT.md)
- The road should feel like it winds through countryside then climbs toward the city
- Use cubic Bezier curves (C commands) for smooth, natural-looking bends
- Example shape: start left-bottom, curve up-right through hills, dip slightly at midpoint, climb to city on upper-right

**STOP_OFFSETS:** 5 percentage string values representing where each of the 5 stops falls along the path as a percentage of total path length:
- Stop 0 (start): `"5%"` — near the start of the road
- Stop 1: approximately `"28%"` — first waypoint
- Stop 2: approximately `"52%"` — midpoint
- Stop 3: approximately `"75%"` — third waypoint
- Stop 4 (end): `"95%"` — near the end of the road
These are design-time constants. Add an inline comment: `// Design-time constants — update together with ROAD_PATH_D if path geometry changes`

**RoadPath component:** Renders a `<path>` SVG element with `ROAD_PATH_D` as its `d` attribute. Style the road to look like a paved road:
- `stroke`: warm tan/beige road color matching Screenshot 1 (approximately `#D4A96A` or `#C8A05A`)
- `strokeWidth`: `14` (visually substantial road, but not overwhelming)
- `fill`: `"none"`
- `strokeLinecap`: `"round"`
- `strokeLinejoin`: `"round"`

The RoadPath component is designed to be rendered inside the `<svg>` element in MapSvg.

```typescript
'use client'

// IMPORTANT: ROAD_PATH_D and STOP_OFFSETS must always be updated together.
// ROAD_PATH_D is used by both the visual <path> in MapSvg AND the car's CSS offsetPath.
// Changing one without the other will cause the car to travel off the visual road.
export const ROAD_PATH_D =
  'M 80,560 C 150,540 200,480 280,450 C 360,420 420,460 500,420 C 580,380 620,320 720,300 C 820,280 880,340 980,300 C 1080,260 1140,220 1220,210 C 1270,205 1300,200 1320,200'

// Stop offsets along the path — percentage of total path length.
// 5 values corresponding to stops 0–4.
// These are design-time estimates; adjust after visual verification in the browser.
export const STOP_OFFSETS: readonly string[] = ['5%', '28%', '52%', '75%', '95%'] as const

export function RoadPath() {
  return (
    <path
      d={ROAD_PATH_D}
      stroke="#C8A05A"
      strokeWidth={14}
      fill="none"
      strokeLinecap="round"
      strokeLinejoin="round"
    />
  )
}
```

Feel free to adjust the exact `d` string to produce a more naturally winding, visually beautiful road. The key constraints are: starts lower-left, ends upper-right, has at least 3 meaningful curves, covers the full width of the 1400×800 viewBox.
